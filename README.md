# ML Library in C++

**Updated:** This repo contains 7 models and some evaluation methods built from scratch:
* **Models:** Linear Regression, Logistic Regression, K-Nearest Neighbors, K-Means, Decision Trees, Random Forests, and Multi-Layer Perceptrons.
* **Evaluation & Validation:** RMSE, MAE, R-Squared, Confusion Matrix, Precision, Recall, F1 Score, K-Fold Cross-Validation, Early Stopping, and OOB (Out-Of-Bag) Error.
